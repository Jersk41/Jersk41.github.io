Penjelasan

Aplikasi Todo List merupakan aplikasi untuk mencatat tugas/agenda(todo) yang akan dikerjakan.
dilengkapi dengan sistem login dan registrasi, sehingga tiap user tidak akan melihan task/tugas user lain walaupun masih dalam handphone yang sama.
Aplikasi ini dibuat menggunakan bahasa pemrograman Java, lalu dibantu dengan library Room dan SQLite untuk database.
Dalam pembuatannya pun mengambil referensi dari dokumen Android Developer dan referensi tutorial di youtube dengan bimbingan dari guru-guru SMK Al-Falah.
