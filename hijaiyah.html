Penjelasan

Aplikasi Hijaiyah Search Engine ini dibuat untuk mempermudah santri atau pun guru/ustadz/pengajar
dalam mempelajari tajwid khususnya makhorijul(tempat keluar) huruf beserta sifat-sifat nya.
Aplikasi ini dibuat menggunakan bahasa pemrograman PHP dan framework CodeIgniter untuk sistem Backend serta framework Bootstrap untuk Frontend
pengerjaan aplikasi ini dibuat bersama tim dengan beranggotakan 4 orang.
Saya sebagai Full Stack Developer
Alif Ramadhan dan Kartika Sebagai Designer UI/UX
Imam Hasbi Asadiki sebagai database administrator
dalam pembuatan aplikasi kami juga dibimbing oleh guru-guru di sekolah kami.
