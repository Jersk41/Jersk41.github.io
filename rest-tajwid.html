Aplikasi RESTful API Tajwid

Tajwdi RESTful API

Data mengenai salah satu fan ilmu yaitu tajwid dan lebih khusus menyediakan data seperti makhorijul huruf (tempat keluarnya huruf) beserta sifat-sifat huruf tersebut.
Aplikasi dibuat menggunakan bahasa pemrograman PHP dan framework CodeIgniter.
Kemudian untuk web server sendiri menggunakan Nginx.
Aplikasi ini (REST API) ini merupakan sumber data untuk aplikasi Hijaiyah Search Engine.
